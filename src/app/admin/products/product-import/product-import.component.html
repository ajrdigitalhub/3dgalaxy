Quick Start

  • Upload a Shopify CSV file with products and variant rows.
  • Preview the import before committing to database changes.
  • Choose match mode by SKU or slug.
  • Import products in batches with variant and image upload support.

Supported Shopify Fields

Handle, Title, Body HTML, Vendor, Product Type, Tags, Published, Option1 Name, Option1 Value, Variant SKU, Variant Price, Variant Inventory Qty, Variant Weight, Variant Barcode, Image Src

Other CSV columns are parsed if present and mapped into the import preview groups.
